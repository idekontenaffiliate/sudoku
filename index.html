URL ini dipakai untuk menyinkronkan waktu mulai kedua HP dan mencatat skor akhir. Lihat file AppsScript.gs untuk cara deploy.

Format: 6 karakter acak (0-9, A-Z) + 2 digit jumlah petunjuk awal 17–70. Contoh AB9AAA70 = kode AB9AAA, dibuka 70 angka. AAAAAA17 = kode AAAAAA, dibuka 17 angka (paling sulit). Kedua HP wajib memasukkan kode persis sama.
